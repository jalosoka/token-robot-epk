Artist snapshot

Built for psychedelic floors

Token Robot is a Nighttime Psychedelic project from Switzerland, delivering robotic funk, deep psychedelic grooves, playful sound design and good vibes for psychedelic dancefloors.

Created by Janiv Damti, an experienced Swiss Psytrance artist also known from Silent Sphere, the project combines machine-like movement, high-quality original production and a clear sonic identity.
